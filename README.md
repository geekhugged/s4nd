# s4nd — Startup Simulator

Симулятор для работы над стартап-идеями. Команда опытных фаундеров и YC-партнёров всегда под рукой.

## Быстрый старт

Кинь идею в чат. Или вызови конкретного советника через slash-команду.

**Примеры:**
- `/board` — собери всех советников на boardroom
- `/founders:graham` — Пол Грэм оценит идею
- `/yc:seibel` — Майкл Сейбел разберёт PMF
- `/validators:mom-test` — как правильно спрашивать пользователей
