## 2026-05-30

### Рынок ИИ

**Источник:** Claude (анализ сессии)

- **Инфраструктурный слой закрывается.** Compute, foundation models, cloud APIs — монополизируются OpenAI, Anthropic, Google, Meta. Для стартапа — плохое место для входа.

- **Прикладной слой ещё открыт.** Вертикальные ИИ-решения под конкретные индустрии (юриспруденция, медицина, строительство, логистика) — нет монополий. Workflow automation — замена конкретных процессов, а не "ИИ для всего".

- **Что умирает:** ChatGPT-обёртки без дифференциации коммодитизируются. Общие AI-ассистенты без специфической аудитории теряют позиции.

- **Что растёт:** AI agents (автономные агенты под задачи), data moats (компании с уникальными данными), human-in-the-loop для высоко-рискованных решений.
